# 模块页面设计与功能需求
## 底部导航栏
- 导航至五个模块：首页、集市、+、公告、个人
- 在首页、集市、公告和个人四个模块的首页的底部固定呈现，参考小红书
## 首页
- 包含以下功能
  - 根据选定标签呈现瀑布流的具体内容
  - 搜索 (入口)
  - 动态 (入口)
  - 我的频道，推荐频道，编辑
- 自上而下设计有四个区块
  - 区块一：顶部功能区
    - 分左中右三个子区块
    - 子区块一：搜索 icon
      - 原型是小红书首页右上角的搜索 icon
      - 点击跳转至{搜索}页
    - 子区块二：Logo
      - 1956 | BUCM
    - 子区块三：原生的多功能&退出按钮
  - 区块二：小红书式的标签选项卡 & 标签
    - 参考小红书
    - 默认标签：全部，……
    - 校区标签：良乡，西区，东区
    - 特色标签：四诊，针灸，推拿，方剂，临床，……
    - 更多标签：学习，穿搭，舞蹈，活动，美食，职场，塑形，旅行，音乐，抽象，情感，科普，科研，影视，实习，兼职，……
  - 区块三：小红书式的瀑布流
  - 区块四：底部导航栏
## 搜索 (首页)
- 包含以下功能
  - 返回，搜索框，搜索按钮
  - 历史记录，删除
  - 热搜
## 动态详情页
- 包含以下功能
  - 返回，头像，昵称，关注，分享
  - 图片 (可左右滑动)
  - 内容 (上下)
  - 标签，合计
  - 发布时间，发布地点，不喜欢
  - 评论区
  - 发布评论，点赞，收藏
## 集市
- 模块定位：校园二手市场
  - 只帮助建立联系，不作为交易平台
- 包含以下功能
  - 校区选择 (良乡，和平街，望京)，搜索 (搜索框，入口)
  - 标签选项卡 & 标签
  - 根据选定标签呈现的小红书市集模块式的瀑布流
  - 交易物品 (入口)
## 交易物品页
- 包含以下功能
  - 返回，搜索 (icon，入口)，收藏，分享
  - 头像，昵称，何时来过
  - 理想价
  - 文字描述
  - 图片 (可左右滑动)
  - 留言区
  - 卖方主页 (入口)
  - 发布留言，收藏，联系对方 (入口)
## 联系对方页
- 包含以下功能
  - 返回，昵称，功能 (底部弹窗，入口)
  - 聊天区
  - 语音 (icon)，输入框 (系统键盘，入口)，表情 (icon)，+ (icon，入口：相册，拍摄，发送位置)
## + (发布动态)
- 包含以下功能
  - 上传
  - 添加标题
  - 添加正文或发语音
  - #话题 (底部弹窗，入口) @用户 投票 (底部弹窗，入口)
  - 添加地点
  - 谁可见 (底部弹窗，入口)
  - 高级选项 (页面，入口)
  - 存草稿，发布动态
## 我的
- 包含以下功能
  - 关注，粉丝，获赞，获收藏
  - 动态，评论，收藏，赞过，浏览记录
  - 编辑资料，设置 (入口)
  - 头像，昵称，平台 id与简介 (入口)
  - 校园认证
  - 联系我们
